Watched Cleanup

Watched cleanup removes media after it has been watched for the configured number of days.

Idle Cleanup

Idle cleanup removes media that has not been watched by monitored users for the configured number of days.

Manual Mode

When enabled, Jellycleanerr will continue to track items and move them to the Due list, but nothing will be removed automatically. You can still delete due items manually.
